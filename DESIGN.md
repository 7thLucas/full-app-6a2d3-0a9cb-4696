# Design Guidelines

## Color Palette
- Background: Deep black (#080808)
- Primary Accent: Blood red (#8b0000)
- Secondary Accent: Dark crimson (#6b0000)
- Text/UI: Off-white (#e8e8e8)
- Atmospheric glow: Sickly green (#1a3a1a)

## Typography
- Headings: Horror/Gothic style (e.g. Creepster or similar)
- Body/UI: Clean readable sans-serif

## Visual Style
- Full-screen immersive game canvas
- Minimal HUD to maximize horror atmosphere
- Dark, shadowy environments with limited visibility
- Flickering light effects
- Gothic horror aesthetic throughout

## Game Components
- Full-screen HTML5 canvas game
- Top-down or room-by-room exploration
- Player character navigating dark rooms
- Interactive objects (doors, keys, clues)
- Horror entities/enemies
- Win condition: find and use the exit key
