The Problem

Horror games are locked behind barriers that kill the mood

Downloads, installations, hardware requirements — by the time setup is done, the atmosphere is gone. Players want fear that starts now, not after a 2 GB install and a login screen.

The Solution

One browser link. One bungalow. One objective: get out.

A fully browser-based horror escape game — no friction between the player and the fear. Open the link and you're already trapped inside. Explore room by room, solve the puzzles, survive the horror. The exit exists. Find it.

No Install No Account 15–30 min sessions Any Browser
